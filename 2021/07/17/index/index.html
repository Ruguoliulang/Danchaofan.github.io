hexo-见闻

2021-07-17 | 编程

首先说下hexo 的安装，现在由于github被限制访问，gitee很关键的page闭服更新，所以安装hexo+github时，要多注意自己的网络是不是处在健康状态 ，不然真的会走很多的弯路。

hexo 快速简洁的高效博客框架。

安装nodejs 安装git, nodejs不要用最新版本

使用 Node.js 官方安装程序时，请确保勾选 Add to PATH 选项（默认已勾选）

在你想要的文件位置打开git bash

绑定git和github

生成ssh 密钥

然后yes回车回车回车，生成好的.ssh文件在本地用户文件夹里面选择id_rsa.pub 复制里的内容

粘贴到github里面的ssh key ,标题可以随便写，然后点击add

检查是否链接成功: c出现hi就说明成功了，有的时候会让你输入一个yes

安装hexo之后,（后面的命令都是在刚刚创建的blog中）

检测是否安装成功

安装之后还会出现两种情况，一预览时是空白页，这种情况非常难遇见，大部分是安装的时候下载的依赖不全，检查好网络，重新安装下。二是预览时是好的 ，deloyer之后是空白的，我经历的还是因为网络的问题，下载的依赖不全，重新下载个几次就好了。

推送网站

登录github 新建仓库 在code里复制ssh 放进blog 根目录的config里面。

安装git部署软件

然后

域名绑定

首先购买域名，我用的是阿里云， 解析记录写 Repository name里面设置的名字。source下面创建CNAME注意不要后缀，在里面写上你的域名，然后登录github 在仓库的设置里面找到domain，在里面写入你的域名。重新提交输入域名就可以了。
